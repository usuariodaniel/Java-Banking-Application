# Java Banking Application
## Introduction
This Java project is a simple banking application that utilizes the JOptionPane class from the Swing library to create a graphical user interface (GUI) for user interaction. The application allows users to perform basic banking operations such as checking the balance, depositing funds, and transferring funds between accounts. It's designed to be a straightforward example of using Swing for GUI development in Java.

## Features
- Account Management: Users can manage their account balance through deposits and transfers.
- User Interface: A simple and intuitive GUI using JOptionPane for user interaction.


- Operations:
1. Check balance
2. Deposit funds
3. Transfer funds
4. Exit the application

## Getting Started
### Prerequisites
- Java Development Kit (JDK) 8 or later
- A Java IDE (e.g., Eclipse, IntelliJ IDEA)

### Installation
- Clone the repository to your local machine.
- Open the project in your preferred Java IDE.
- Run the Main class to start the application.

## Usage
Upon running the application, a dialog box will appear displaying the user's account details and the available operations. Users can select an operation by entering the corresponding number and following the prompts.
